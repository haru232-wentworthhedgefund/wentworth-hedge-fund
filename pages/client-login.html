ACCESS POLICY

Private access should be controlled, encrypted, and auditable.

For a real client portal, Wentworth Hedge Fund would require secure identity verification, encrypted session handling, permission-based document access, and operational monitoring. A static GitHub Pages website is appropriate for presentation and public information, but it should not be used to collect passwords, manage client accounts, or distribute confidential investment materials.

Future implementation should be supported by a secure backend application, multi-factor authentication, HTTPS-only deployment, and compliance review before any confidential client information is made available online.
